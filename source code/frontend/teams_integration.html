How it works

Setup Guide

Follow these three steps to get Teams integration ready when the feature launches.

1

Register App in Azure AD Portal

Sign into portal.azure.com → Azure Active Directory → App Registrations → New Registration. Give your app a name, set the redirect URI, and note down the Client ID and Tenant ID.

2

Configure Credentials in Settings

Generate a Client Secret under Certificates & Secrets in Azure. Then paste the API Key, Tenant ID, and Client ID into the configuration form below and hit Save.

3

Connect and Fetch Meetings

Click Test Connection to verify your credentials against the MOM Generator API. Once confirmed, the integration will auto-discover your Teams calendar and available meeting recordings.
